# Contributing

## We're Building

GameStake is a sustainable gaming rewards protocol. We need help.

## Open Roles

### Solana Developer

**Requirements**:
- Anchor framework experience
- CPI integration (Jupiter Earn)
- Security-first mindset

**Tasks**:
- Treasury PDA implementation
- Jupiter Earn staking integration
- Player registry and distribution logic

### Frontend Developer

**Requirements**:
- React/TypeScript
- Web3 wallet integration
- Dashboard/analytics experience

**Tasks**:
- Admin dashboard
- Player rewards UI
- SDK for game integration

### Game Partnerships

**Requirements**:
- Connections to Solana gaming projects
- Understanding of Web3 gaming economics
- BD/partnerships experience

**Tasks**:
- Onboard pilot games (Block Stranding, Bonk Arena, etc.)
- PSG1 GameHub integration coordination
- Community building

## How to Contribute

1. Read the [WHITEPAPER.md](WHITEPAPER.md)
2. Check GitHub Issues for open tasks
3. Fork, implement, submit PR
4. Or reach out directly: LinuxMakesMeHappy@proton.me

## Compensation

- **Pre-funding**: Contributor tokens (vested)
- **Post-funding**: Paid roles available

## Code Standards

- Rust/Anchor for smart contracts
- TypeScript for SDK/frontend
- Tests required
- Security review before merge
